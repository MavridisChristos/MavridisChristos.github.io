Selected Projects

Hybrid Systems

Real-Time Switched System Identification

  • Description: Real-time identification method for discrete-time switched affine models based on a system of adaptive algorithms running in two timescales.
  • Details: A stochastic approximation algorithm based on online deterministic annealing estimates the partition of the space that defines the switching signal, as well as the number of necessary modes. At the same time, recursive identification running at a higher timescale updates the parameters of the local models.
  • Keywords: Switched System Identification, Hybrid Systems, Online Deterministic Annealing.
  • Links: (CDC23) (ECC24) (TAC22) (TNNLS22) (GitHub)

Cyber-Physical Systems Security

CPS Security in Dynamic Games under Cognitive Hierarchy

  • Description: Identification of the profiles and capabilities of attackers injecting adversarial inputs to a cyber-physical system.
  • Details: Principles of behavioral game theory – specifically the concept of level-k thinking – is employed to construct a database of potential attack vectors. By observing the state trajectories under sequential interactions with different adversaries, the defender adaptively estimates both the number and the profiles of the different attack signals using an online deterministic annealing approach.
  • Keywords: CPS Security, Bounded Rationality, Attack Identification
  • Links: (IFAC23) (ECC24) (CDC24)

Communication-Aware Control

Funded by the Swedish Foundation for Strategic Research and Ericsson AB.

Hierarchical Communication-Aware Motion Planning

  • Description: A novel communication-aware motion planning architecture that aims to improve the performance and robustness of the operation of an interacting team of agents communicating over a 5G wireless network.
  • Details: Multi-agent planning based on a hierarchical modeling and adaptation method that identifies and continuously updates a hybrid network digital twin model describing the properties of a 5G network and its environment.
  • Keywords: Communication-Aware Planning, Hybrid Network Digital Twin, Hierarchical Learning

Past Projects

Human-Robot Interaction & Collaboration

A Human-Robot Interface based on Surface EEG Sensors

  • Description: A Brain-Machine Interface decoding visual attention into montion in 3D space.
  • Details: Suitable for robot teleoperation or as part of a shared-control robotic manipulation scheme. No restrictions imposed on blinking, head movement, etc. Appropriate for upper-extremity amputees.
  • Keywords: Brain-Machine Interface, Human-Robot Interaction, EEG-based Control.
  • Links: (IROS20)

Safe robot navigation with spatial and temporal constraints

  • Description: Robot Navigation Under MITL Constraints Using Time-Dependent Vector Field Based Control.
  • Details: Translating rudimental MITL formulas to control problems in the state space. Automatic design of a real-time controller as a hybrid automaton based on appropriately defined time-dependent vector fields.
  • Keywords: Safe Robot Navigation, Temporal Logic, Prescribed Performance Control.
  • Links: (CDC19)

Human-Robot Collaboration based on Motion Intention

  • Description: Physical Human-Robot Collaboration (HRC) scheme with adaptive impedance control under unknown robot dynamics.
  • Details: Human intention estimation through prescribed performance adaptive estimation law. No explicit communication, use of position, velocity and force/torque feedback.
  • Keywords: Human-Robot Collaboration, Human Intention Estimation, Prescribed Performance Control.
  • Links: (ECC18)

Intelligent Transportation

Real-Time Priority-Based Cooperative Highway Merging for Heterogeneous Autonomous Traffic

  • Description: A novel cooperative highway merging method taking into account heterogeneous, in terms of physical properties and priority, autonomous vehicles.
  • Details: Quadratic programming optimization with safety guarantees operating in real-time.
  • Keywords: Highway Merging, Mixed Traffic, Heterogeneous Autonomous Traffic Control.
